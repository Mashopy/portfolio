Site de dons pour la Commune de Balot


Projet réalisé dans le cadre d’un stage obligatoire d’une durée d’environ de 2 mois pour la Commune de Balot (21130).

Le but de ce projet était de créer un site permettant de lister les différents projets de la commune, un espace admin pour que la mairie de Balot puisse directement gérer le contenu du site et d’intégrer les différents moyens de paiement pour aider la commune.

Le site a été réalisé en HTML, CSS, JS, SQL et avec le framework Node.Js

AOSP pour le Nothing Phone (2a)


Projet en réalisé et maintenu depuis Juin 2025. Le but de ce projet était de réaliser un portage stable d’AOSP pour offrir aux utilisateurs de cet appareil une version plus épurée, libre, personnalisable et constament mise à jour.

Ce projet a été réalisé avec du C, C++, Python, Kotlin et un important travail d’ingénierie inverse pour adapter le système Android opensource aux spécificités de l’appareil.

Le code source est disponible directement sur GitHub.
